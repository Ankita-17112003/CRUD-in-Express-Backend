# Express CRUD Backend API

This project is a simple **CRUD (Create, Read, Update, Delete)** backend built using **Node.js** and **Express.js**.
It demonstrates how to build RESTful APIs for managing resources.

---

## 📦 Features

- Create a new resource
- Read all resources or a single resource
- Update an existing resource
- Delete a resource
- RESTful API structure
- Middleware support (JSON parsing, error handling)

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
